# 🔄 X-UI to Marzban Migration Script

### انتقال خودکار کاربران از X-UI به Marzban

![Version](https://img.shields.io/badge/version-2.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Python](https://img.shields.io/badge/python-3.6+-yellow.svg)

---

## ✨ ویژگی‌ها

* 🚀 اجرای سریع تنها با یک دستور
* 🤖 شناسایی خودکار دیتابیس X-UI و کاربران
* 📊 محاسبه دقیق حجم باقی‌مانده کاربران
* 🔐 پشتیبانی کامل از پروتکل VLESS
* 🛡️ انتقال امن با بررسی صحت اتصال
* 📝 ارائه گزارش کامل از کاربران منتقل‌شده

---

## 🎯 منطق انتقال کاربران

| وضعیت کاربر              | حجم باقی‌مانده | نتیجه انتقال        |
| ------------------------ | -------------- | ------------------- |
| ✅ فعال و نامحدود         | ∞              | نامحدود             |
| ✅ فعال با حجم باقی‌مانده | بیشتر از 0     | همان حجم باقی‌مانده |
| ✅ فعال با حجم تمام‌شده   | 0 GB           | 1 GB                |
| ❌ غیرفعال                | -              | منتقل نمی‌شود       |

---

## 📋 پیش‌نیازها

> نیازی به نصب دستی وابستگی‌ها نیست؛ اسکریپت به‌صورت خودکار همه موارد موردنیاز را نصب و تنظیم می‌کند.

* Ubuntu / Debian
* دسترسی Root
* نصب و فعال بودن X-UI (نسخه سنایی)
* نصب و فعال بودن Marzban
* دسترسی به اینترنت

---

## 🚀 نصب و اجرا

```bash
bash <(curl -Ls https://raw.githubusercontent.com/JavadWolf-af/X-ui-To-Marzban/main/install.sh)
```

## 📈 گزارش انتقال

پس از پایان فرآیند، اسکریپت اطلاعات زیر را نمایش می‌دهد:

- تعداد کاربران منتقل‌شده
- تعداد کاربران ناموفق
- حجم باقی‌مانده هر کاربر
- وضعیت نهایی انتقال
- خطاهای احتمالی

## 🛠️ عیب‌یابی

| مشکل                      | راه‌حل                                      |
| ------------------------- | ------------------------------------------- |
| فایل دیتابیس پیدا نشد     | فایل `x-ui.db` را در مسیر صحیح قرار دهید    |
| خطا در اتصال به مرزبان    | آدرس، پورت و اطلاعات ورود را بررسی کنید     |
| اینباند VLESS پیدا نشد    | ابتدا یک اینباند VLESS در مرزبان ایجاد کنید |
| خطای 500 هنگام ساخت کاربر | معمولاً موقتی است؛ مجدداً تلاش کنید         |
